# LeadFlowAI

AI-powered lead generation SaaS app built with React + Vite + Tailwind CSS.

## Features

- 🔍 AI Lead Finder with smart discovery
- ⚡ Lead scoring and qualification
- ✉️ Personalized outreach automation
- 📊 Real-time analytics dashboard
- 🔗 CRM integrations (Salesforce, HubSpot, Pipedrive, etc.)
- 🌍 Multi-language support
- 🔒 GDPR compliant

## Tech Stack

- React 18 + Vite
- React Router v6
- Tailwind CSS 3
- Dark theme UI

## Getting Started

```bash
npm install
npm run dev
```

## Pages

- `/` — Homepage with hero, features, testimonials, FAQ
- `/ai-lead-finder` — Interactive lead search with mock results
- `/crm-integrations` — CRM and tool integration cards
- `/pricing` — Monthly/yearly pricing toggle
- `/get-started` — Signup form
